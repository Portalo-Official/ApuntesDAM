
# Ejecutar js

Teniendo node.js instalado, podemos ejecutar un archivo JS en nuestra terminal.

Solo hace falta poner el comando `{bash}node` mas el nombre del archivo:

```js title='app.js'
let nombre = 'Fernando';
console.log(nombre);
  
nombre = 'Juan';
console.log(nombre);
```

El comando: `{bash}node app.js` ejecutara el archivo.
